## What is This?

Paperclip is an open-source control plane for AI-agent companies. It coordinates teams of AI agents — Claude, Codex, Cursor, Gemini, and others — to run as an autonomous business under human oversight.

This documentation analyzes Paperclip's **orchestrator layer**: how work gets created, routed, assigned, tracked, governed, and completed. It was written as research for building XO Org's own orchestrator.
